주차 위젯 앱 개인정보처리방침

요약: 주차 위젯 앱은 사용자의 개인정보를 수집하지 않으며, 모든 데이터는 사용자의 기기에만 저장됩니다.

1. 개인정보의 처리 목적

주차 위젯 앱("앱")은 사용자가 주차 위치를 기록하고 위젯을 통해 확인할 수 있는 서비스를 제공합니다. 본 앱은 개인정보를 수집하지 않습니다.

2. 개인정보의 처리 및 보유기간

본 앱은 다음과 같이 정보를 처리합니다:

  • 주차 위치 정보: 사용자가 입력한 주차 위치는 오직 사용자의 기기에만 저장됩니다
  • 저장 시간: 주차 위치가 저장된 시간은 기기에만 저장됩니다
  • 보유기간: 사용자가 앱을 삭제하거나 데이터를 직접 삭제할 때까지

3. 개인정보의 제3자 제공

본 앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다. 모든 데이터는 사용자의 기기에서만 처리됩니다.

4. 개인정보처리 위탁

본 앱은 개인정보 처리를 위탁하지 않습니다.

5. 정보주체의 권리·의무 및 그 행사방법

사용자는 언제든지 다음과 같은 권리를 행사할 수 있습니다:

  • 저장된 주차 위치 정보 확인
  • 저장된 정보 수정
  • 저장된 정보 삭제
  • 앱 삭제를 통한 모든 데이터 완전 삭제

6. 개인정보의 안전성 확보조치

본 앱은 다음과 같은 기술적 조치를 통해 개인정보를 보호합니다:

  • 모든 데이터는 기기 내부 저장소에만 보관
  • 외부 서버로 데이터 전송 없음
  • 암호화된 로컬 저장소 사용

7. 권한 사용 내역

본 앱은 다음 권한을 사용합니다:

  • 인터넷 권한: React Native 프레임워크 요구사항 (실제로 인터넷 연결 사용 안함)
  • 위젯 권한: 홈 화면 위젯 표시를 위함
  • 알람 권한: 자정 위젯 업데이트를 위한 스케줄링

8. 개인정보보호책임자

Parking Widget App Privacy Policy

Summary: The Parking Widget App does not collect personal information and all data is stored only on the user's device.

1. Purpose of Personal Information Processing

The Parking Widget App ("App") provides a service that allows users to record parking locations and check them through widgets. This app does not collect personal information.

2. Personal Information Processing and Retention Period

This app processes information as follows:

  • Parking Location Information: Parking locations entered by users are stored only on the user's device
  • Save Time: The time when parking locations were saved is stored only on the device
  • Retention Period: Until the user deletes the app or directly deletes the data

3. Provision of Personal Information to Third Parties

This app does not provide users' personal information to third parties. All data is processed only on the user's device.

4. Personal Information Processing Consignment

This app does not consign personal information processing.

5. Rights and Obligations of Data Subjects and How to Exercise Them

Users can exercise the following rights at any time:

  • Check stored parking location information
  • Modify stored information
  • Delete stored information
  • Complete deletion of all data by uninstalling the app

6. Measures to Ensure the Safety of Personal Information

This app protects personal information through the following technical measures:

  • All data is stored only in the device's internal storage
  • No data transmission to external servers
  • Use of encrypted local storage

7. Permission Usage

This app uses the following permissions:

  • Internet Permission: React Native framework requirement (does not actually use internet connection)
  • Widget Permission: For displaying home screen widgets
  • Alarm Permission: For scheduling midnight widget updates

8. Personal Information Protection Officer

Personal Information Protection Officer: Developer

Contact: Available through developer email

Notification of Policy Changes: Announced through this page

9. Changes to Privacy Policy

This privacy policy applies from the effective date, and when there are additions, deletions, and corrections of changes according to laws and policies, it will be notified through announcements 7 days before the implementation of changes.
